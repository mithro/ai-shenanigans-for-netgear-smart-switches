New Features:

  • Can enable and disable the power saving mode (IEEE 802.3az) on the Switch Information page. (In the device UI, select System > Management > Switch Information.)

Bug Fixes:

  • Fixes the issue where the settings for the 802.1Q VLAN Advanced function (in the device UI, select VLAN > 802.1Q > Advanced) are missing in the German version.

Known Issues:

  • When you access the device UI, all HTTP packets are forwarded to the embedded CPU. If flow control is enabled, the throughput is impacted if the rate at which HTTP packets are forwarded is higher than the CPU can manage.
  • A configuration that is obtained from the Plus Utility cannot be used by device UI, and the other way around.
  • The time-out period of a dynamic IGMP router port is not exactly 300 seconds but could be anywhere between 300 and 600 seconds.
  • The time-out period of an IGMP host is not exactly 260 seconds but could be anywhere between 260 and 520 seconds.

Warnings:

Read and follow these warnings before updating your firmware:

  • After you upgrade the GS108Ev3 switch to firmware version 2.06.03 or a later version, the switch cannot operate with a Plus Utility version earlier than version 2.7.2.  Download the latest Plus Utility (version 2.7.8) by visiting http://www.netgear.com/support/product/PCU.
  • Firmware version 2.06.24 includes three firmware images. Each image supports a different language for the device UI:  English, German, or Japanese. The supported language is identified by the suffix of the image filename:
  1. EN - English.
  2. GR - German (Deutsch).
  3. JP - Japanese (日文).
