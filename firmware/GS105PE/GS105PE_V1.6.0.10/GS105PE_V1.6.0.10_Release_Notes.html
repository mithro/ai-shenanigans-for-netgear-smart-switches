New Features:

  • Added a control option to disable or enable file transfer via TFTP (Trivial File Transfer Protocol), under System > Management > Switch Managed Mode). Default is Disable. 

Security Fixes: 

  • Various security vulnerability fixes and enhancements.

Firmware Update Instructions:

Important: Read and follow these instructions before updating your firmware.

  • When the GS105PE is upgraded to firmware version V1.5.0.4 or higher, it cannot work with any Plus Utility version prior to V2.4.3.  Please download the latest Plus Utility (v2.7.8) from http://www.netgear.com/support/product/PCU 

To update your product’s firmware, follow the instructions in your product’s user manual. To find your user manual, visit https://www.netgear.com/support/, enter your model number in the search box, and click the Documentation button on the product page.
