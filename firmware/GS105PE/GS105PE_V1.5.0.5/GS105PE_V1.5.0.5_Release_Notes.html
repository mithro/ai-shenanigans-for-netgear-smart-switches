Fixes and Modifications:

  • Fixed the issue that there is a chance where link cannot be established with random port after power cycle

WARNING:

  1. Once GS105PE FW is upgraded to V1.5.0.4, it is imperative to upgrade the bootloader also.   Please login to switch using web GUI.  V1.5.0.4 FW will prompt you automatically for the required bootloader version.  Please make sure you pick the right version.  Incorrect version will cause the switch to become inoperable. 
  2. Once you have upgraded both FW and bootloader, please perform a factory default for the fix to take effect.
  3. After GS105PE is upgraded to FW version V1.5.0.4 or higher, it will not work with Plus Utility version prior to V2.4.3.  Please download the latest Plus Utility from http://www.netgear.com/support/product/PCU

Upgrade Procedure:

  1. Download the new firmware 1.5.0.4 from NETGEAR product support Website and store it in local PC, and remember its location in the PC’s file system. Unzip it to get the switch firmware image “GS105PE_V1.5.0.5.bin”.

    Download Link: http://www.downloads.netgear.com/files/GDC/GS105PE/GS105PE_V1.5.0.5.zip

  2. There are two methods to upgrade the new firmware from PC to the switch.
    • HTTP
    • ProSAFE Plus Utility

    Below are the steps to upgrade the firmware via HTTP, which is the most convenient method. For the other method, please check the IG and SW User Manual for details.

  3. To use HTTP method, go to GUI page “Maintenance > Firmware Upgrade".
  4. Click Enter Loader Mode button and it will go to Firmware Upgrade page.
  5. Click the Browse button and locate and select the new firmware image file “GS105PE_V1.5.0.5.bin”.
  6. Click the Apply button.
  7. When the upgrade is complete, your switch restarts. The upgrade process typically takes about three minutes.
  8. Once FW upgrading completes, please login to switch by web GUI to upgrade boot loader.
  9. When login to the switch, it will go to Bootloader Upgrade page.
  10. It will show which boot loader version you should use and click Browse button and locate and select the correct boot loader version and click APPLY.
  11. Once boot loader upgrading is complete, the switch will go back to login page.

WARNING:

To avoid the risk of corrupting the firmware, do not interrupt the upgrade. For example, do not turn off the switch or disconnect it.
