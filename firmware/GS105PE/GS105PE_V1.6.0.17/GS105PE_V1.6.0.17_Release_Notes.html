Bug Fixes:

  • Fixes the issue where port 5 incorrectly adjusts PoE power requests from the powered device (PD).
  • Fixes the issue where a subnet mask length set below 24 causes the switch to reject IP addresses using the following patterns: x.x.x.255 and x.x.x.0. Example: If the subnet mask length is IP address/22, it rejects 172.16.16.255/22 and 172.16.17.0/22. 

Security Fixes:

  • This firmware addresses security vulnerabilities. For more information about security vulnerabilities, visit https://www.netgear.com/about/security.

Warnings:

Read and follow these warnings before updating your firmware:

  • You cannot restore the configuration file on a GS105PE switch that runs an earlier firmware version than the version from which you saved the configuration file.
  • When you update the GS105PE switch to firmware version 1.5.0.4 or a later version, the switch cannot operate with a Plus Utility earlier than version 2.4.3.  You can download the latest Plus Utility (version 2.7.8) from the Plus Utility support page: http://www.netgear.com/support/product/PCU

Firmware Update Instructions:

To update your product’s firmware, follow the instructions in your product’s user manual. To find your user manual, visit https://www.netgear.com/support/, enter your model number in the search box, and click the Documentation button on the product page.
